# LoongEgg.LoongLog
一个简单实用的跨平台Logger工具，欢迎体验！A cross platform .NET C# log, contains debug logger, console logger, and file logger.Enjoy it! 
# 怎么获得
除了直接下载源码编译，现在可以直接在Nuget搜索LoongEgg.LoongLog安装
# TargetFramework 
.NET Standard 2.0
> check here https://docs.microsoft.com/en-us/dotnet/standard/net-standard
