# LoongEgg.LoongLog
一个简单实用的跨平台Logger工具，欢迎体验！   
A cross platform .NET C# log, contains debug logger, console logger, and file logger.   
Enjoy it! 
## Feature
- Colorfull Console log output
- Eafy to use
## 怎么获得
除了直接下载源码编译，现在可以直接在Nuget搜索LoongEgg.LoongLog安装
## TargetFramework 
.NET Standard 2.0
> check here https://docs.microsoft.com/en-us/dotnet/standard/net-standard
## Minimum platform versions
- .Net Core 2.0 [Checked]
- .Net Framework 4.6.1 [Checked]
- Xamarin.iOS 10.14
- Xamarin.Mac 3.8
- Xamarin.Android 8.0[Checked]
